EntityExtractor¶

class rs_helper.core.EntityExtractor.EntityExtractor(text: str)[source]¶

Bases: object

General Class to perform Named Entity Recognition on new problem statements provided in the system. Entities will be extracted by the pattern:

CHUNK: {<NN.*><NN.*>} CHUNK: {<DT.*>?<NN.*>} CHUNK: {<V.*><N.*>+}

and then sorted by (Position Index + Term frequency) / n_words
